# Resources Upload Directory

This folder is for **admin use only** — upload your freebies here.

## Folder Structure

- `guides/` — PDF guides, markdown files, tutorial notes
- `templates/` — Notion templates, workflow exports, config files  
- `tools/` — Spreadsheets, calculators, scripts
- `videos/` — Video walkthroughs, screen recordings
- `starter-kit/` — All files for the AI Automation Starter Kit bundle

## How to Use

1. Drop your resource files into the appropriate subfolder
2. Update the download links in `resources.html` to point to the file path, e.g.:
   ```html
   <a href="resources/guides/n8n-workflow-notes.pdf" class="btn btn-secondary btn-download">Download Free</a>
   ```
3. For external links (Notion, Google Sheets), use the full URL directly

## Notes
- Keep file names lowercase with hyphens (e.g., `voice-agent-blueprint.pdf`)
- This folder is served statically — anything here is publicly accessible
- Do NOT put private/draft files here
